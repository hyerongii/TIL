
#  CLI

### 문법 및 활용
- 현재 디렉토리
```cli
cd.
```
- 현재의 상위 디렉토리
```cli
cd..
```
- 디렉토리로 이동
```cli
cd 파일명/
```
- 파일 생성
```cli
touch 파일명.확장자
```
- 디렉토리 생성
```cli
mkdir 파일명.확장자
```
- 현재 작업중인 디렉토리 내부의 폴더/파일 목록 출력
```cli
ls
ls -a   # 숨김 폴더/파일 표시
```
- 폴더/파일 열기
```cli
start 파일명.확장자
```
- 파일 삭제
```cli
rm 파일명.확장자
rm -r 디렉토리명
```
- 코드로 열기
```cli
code 파일명.확장자
code . (현재 위치 코드로 열기)
```

- 절대 경로
```cli
pwd
```
